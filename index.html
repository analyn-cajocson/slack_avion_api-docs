User Registration

HTTP Method: POST
URL: {{url}}/api/v1/auth/
Sample Request Body
{
    "email": "user1@example.com",
    "password": "12345678",
    "password_confirmation": "12345678"
}
Parameters
Name Description Required
email User email Yes
password User password Yes
password_confirmation Re-type password Yes

Send Message

HTTP Method: POST
URL: {{url}}/api/v1/messages
Sample Request Body
{
    "receiver_id": 1,
    "receiver_class": "User",
    "body": "kamusta?"
}
Parameters
Name Description Required
receiver_id ID of the message's receiver Yes
receiver_class Type of the receiver. User for direct message, Channel for sending a message in a channel Yes
body Message body Yes
Request Headers

Get these values from the login response header

Name Required
access-token Yes
client Yes
expiry Yes
uid Yes
